# EARNED Calculator

Financial control for commission professionals.

## Quick Deploy to Vercel

1. Click the button below to deploy:
   [![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USERNAME/earned-calculator)

2. Or manually:
   - Fork this repo
   - Go to vercel.com
   - Click "New Project"
   - Import this repository
   - Click "Deploy"

## Local Development

```bash
npm install
npm run dev
```

## Features

- Commission calculator with multiple split types
- Brokerage comparison
- Annual projections
- Smart money allocation
- Cap system tracking

Built with React + Vite + Tailwind CSS
